# ArmA FOV Changer

A simple and lightweight utility to adjust the Field of View (FOV) in ArmA 3.

## Installation

1. Navigate to the [Releases](https://github.com/Eelguor/ArmA-FOV-changer/releases) page.
2. Download the latest `.exe` file.
3. Place the executable in a convenient location (no installation required).

> **Note:** Windows SmartScreen may flag the executable because it is an unsigned application. If prompted, click **More info** -> **Run anyway**.

## Usage

1. Make sure ArmA 3 is closed.
2. Open `ArmA-FOV-changer.exe`.
3. Select your ArmA 3 profile and then the corresponding `.ArmA3Profile` file.
4. Input your desired FOV value.
5. Click apply.
